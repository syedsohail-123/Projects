Crypto Price Tracker API - Documentation
 Features- Get real-time price of any cryptocurrency by CoinGecko ID- Supports multiple fiat currencies (default: USD)- Built with FastAPI for high performance and easy documentation
 Tech Stack- FastAPI  Web framework- Python Requests  To fetch data from CoinGecko API- CoinGecko API  Public crypto price source

 API Endpoints
Crypto Price Tracker API - Documentation
 1. Home Route (GET /):
   Response: {"msg": "Welcome to Crypto Price Tracker API"}
 2. Get Crypto Price (GET /price/{coin}?currency=usd):
   - Path Parameter: coin (e.g., bitcoin)
   - Query Parameter: currency (optional, default: usd)
   Example: /price/bitcoin?currency=inr
   Response: {"coin": "bitcoin", "currency": "inr", "price": 2543667}
   Error: {"error": "Coin 'fakecoin' not found."}
    CoinGecko Reference
 Check valid coin IDs: https://www.coingecko.com/en/api/documentation
